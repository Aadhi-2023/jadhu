

# Pet Breeder Website

A clean, modern, and responsive website for a pet breeder business.

## Pages

### 1. Home Page
- Hero section with a welcoming headline and call-to-action button
- "About Us" section highlighting years of experience, mission, and values
- Featured breeds preview (3-4 cards linking to the breeds page)
- Testimonials section from happy pet owners

### 2. Breed Categories Page
- Grid of breed cards with images, breed names, and short descriptions
- Categories for different pet types (e.g., dogs, cats, or specific breeds)
- Each card shows key traits like temperament and size

### 3. Gallery Page
- Masonry-style photo grid showcasing pets
- Filter tabs by breed or category
- Lightbox view when clicking on images

### 4. Contact Page
- Contact information: email, phone number, and address
- Contact form (name, email, subject, message)
- Embedded map placeholder showing location
- Business hours section

### 5. Shared Layout
- **Navigation bar** — Logo + links to all pages, mobile hamburger menu
- **Footer** — Social media links (Facebook, Instagram, Twitter), quick nav links, copyright notice

## Design
- Clean and modern aesthetic with warm, inviting colors
- Soft earthy tones with pops of color for CTAs
- Rounded cards and smooth hover animations
- Fully responsive across desktop, tablet, and mobile
